# Playwright Setup for Marketplace E2E Testing

## 1. Install Playwright
Run this command in your project root:

npm install -D playwright

## 2. Initialize Playwright
Run:

npx playwright init

This creates a `tests/` folder and config files.

## 3. Run Example Test
After setup, you can run tests with:

npx playwright test
